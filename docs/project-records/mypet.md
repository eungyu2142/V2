# mypet 작업 기록

## 역할

마이펫 등록, 수정, 삭제, 종 선택, 반려동물 정보, 사진, 몸무게, 생일, 입양일 등 마이펫 관련 작업을 기록한다.

## 기준

- 리뷰 작성 시 마이펫에 등록된 반려동물 정보가 선택지로 연결되어야 한다.
- 질문 작성과 기록 작성도 반려동물 기록과 연결되어야 한다.

## 2026-07-03 기록

### 문제 인식

사용자는 하단 내비게이션의 마이펫 기능을 별도 기록 파일로 관리하길 요청했다.

### 해결 과정

`docs/project-records/mypet.md`를 만들고 마이펫 관련 완료 내용과 문제 해결 과정을 누적 기록하도록 했다.

### 완료 기록

마이펫 작업 기록 파일을 생성했다.
