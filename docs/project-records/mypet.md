# 마이 펫 작업 기록

## 기능 기준

- 화면 명칭은 `마이 펫`으로 통일한다.
- 오른쪽 아래 `+` 버튼으로 펫 등록을 시작한다.
- 등록 화면 왼쪽 위에는 닫기 버튼 대신 `←` 뒤로가기 버튼을 둔다.
- 다음 단계로 이동한 뒤에도 이전 단계로 돌아갈 수 있어야 한다.
- 필수 항목은 종류, 종, 성별, 이름이다.
- 사진, 체중, 생일, 입양일은 선택 항목이며 값을 입력하지 않아도 다음 단계로 이동할 수 있다.

## 등록 순서

1. 종류: 파충류, 조류, 설치류, 양서류, 기타
2. 종: 종류에 맞는 고정 선택지
3. 성별: 수컷, 암컷, 미구분
4. 이름(닉네임), 사진
5. 체중(선택), 단위 g/kg
6. 생일(선택), 달력
7. 입양일(선택), 달력
8. 최종 확인 및 등록 완료

## 종 선택지

- 파충류: 개코, 비어디드래곤, 이구아나, 카멜레온, 왕도마뱀, 스킨크, 육지 거북, 수생 습지 거북, 콘스네이크, 킹스네이크, 볼파이톤, 보아-파이톤, 호그노즈, 기타 도마뱀, 기타 뱀, 기타
- 조류: 앵무새, 닭, 기타 조류, 기타
- 설치류: 슈가글라이더, 고슴도치, 햄스터, 기타 설치류, 기타
- 양서류: 팩맨, 트리프록, 두꺼비(토드), 뉴트, 살라만다, 아홀로틀, 기타 양서류, 기타
- 기타: 기타

## 목록 화면

- 동물 이름, 종, 분류를 검색할 수 있다.
- 전체(기본값), 파충류, 설치류, 양서류, 조류, 기타 필터를 제공한다.
- 필터는 중복 태그를 만들지 않고 한 번에 하나만 선택한다.
- 첨부한 사진이 있으면 비율을 유지해 정사각형 영역에 `object-fit: cover`로 표시한다.
- 사진이 없으면 코드로 만든 동물 분류 아이콘을 표시한다.
- 등록 카드에는 수정, 삭제, `기록으로 이동` 버튼을 제공한다.
- 데스크톱은 카드 2열, 모바일은 1열로 표시해 한 항목이 전체 너비를 차지하지 않게 한다.
- 상단의 별도 `n마리 등록됨` 요약 배너는 사용하지 않는다.

## 데이터 연결

- 펫 데이터는 `exocare-pets` 로컬 저장소에 유지한다.
- 수정 시 기존 ID를 유지하고 동일한 항목을 갱신한다.
- `기록으로 이동`은 기록 탭으로 전환한다.
- 기록 탭의 펫 목록에도 마이 펫 데이터를 전달한다.

## 2026-07-09 초기화 복구

초기화된 마이 펫 탭에서 위 등록 흐름, 선택형 종 목록, 검색과 필터, 사진, 무게 단위, 선택 날짜, 최종 확인, 수정/삭제, 기록 이동, 반응형 2열 카드를 복원했다.

### 확인 중 발생한 오류

- 첫 빌드에서 `DiaryPage`에 필요한 `pets`, `onAddPet` 속성이 빠졌다는 타입 오류가 발생했다.
- 기존 코드에 `formatPetDetails`가 남아 있어 중복 함수 오류가 발생했다.
- 기록 화면에 현재 펫 목록과 펫 추가 동작을 전달하고 중복 함수를 제거해 해결했다.
- 브라우저 검증 중 DOM 스냅샷 기능에서 `incrementalAriaSnapshot is not a function` 오류가 발생했다. 화면 자체 오류는 아니며, 읽기 전용 DOM 검사와 실제 스크린샷으로 대신 검증했다.

### 검증 결과

- `npm run build` 성공
- 마이 펫 탭에서 검색 및 6개 분류 필터 노출 확인
- 펫 등록 첫 화면에서 뒤로가기 화살표, 진행률, 종류 5개 선택지 확인
- 1280px 화면에서 등록 UI가 겹치거나 잘리지 않는 것을 스크린샷으로 확인

## 2026-07-09 분류 필터 수정

- 분류 태그를 높이 약 40px, 아이콘 23px 기준으로 축소했다.
- 초기화 전 펫 데이터가 `group`, `category`, 영문 분류값, 한글 분류명 중 어느 형식으로 저장돼 있어도 표준 분류값으로 변환한다.
- 저장된 분류가 없을 때는 등록된 종이 어느 종 목록에 포함되는지 확인해 분류를 복구한다.
- 따라서 파충류 태그를 선택하면 파충류로 정규화된 펫만 표시된다.

### 확인 중 발생한 오류

- 브라우저 자동 확인에서 마이 펫 버튼은 1개로 확인됐지만 클릭 명령이 `Runtime.evaluate` 시간 초과로 중단됐다.
- 소스 타입 검사와 프로덕션 빌드는 정상 통과했다.
