# Q&A 닉네임 및 기록 게이지 복구

## 요청 요약

- Q&A 목록과 상세에서 `작성자`라는 고정 문구 대신 로그인한 사용자의 닉네임을 표시한다.
- 기록 첨부가 있는 질문 카드의 오른쪽 시각화 게이지바가 사라지지 않도록 복구한다.

## 분석 및 판단

게시글과 댓글의 작성자 표시를 사용자 프로필의 `nickname` 우선 값으로 연결하고, 닉네임이 비어 있을 때만 username과 기본 사용자 문구를 대체값으로 사용했다. 기록 미리보기 막대는 부모 시각화 영역의 고정 높이를 보장해 퍼센트 높이가 계산될 기준을 항상 갖도록 했다.

## 수정 파일

- `src/App.tsx`
- `src/App.css`
- `src/features/diary/DiaryPage.tsx`

## 핵심 변경 내용

- 새 게시글, 새 댓글, 본인 게시글 상세와 목록 카드에 닉네임 표시
- 기존 다른 사용자의 표시명은 유지
- 기록 첨부 카드의 막대 시각화 영역 높이와 막대 최소 표시 보강
- 빌드를 막던 사용되지 않는 다이어리 추천 상수 제거
