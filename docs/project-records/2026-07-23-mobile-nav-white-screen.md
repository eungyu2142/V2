# 2026-07-23 모바일 내비바 흰 화면 수정

## 요청 요약

모바일 링크에서 다른 하단 내비바로 들어가도 흰 화면이 아니라 각 화면이 정상적으로 보이도록 수정해 달라는 요청.

## 분석·판단 이유

하단 내비바는 `App.tsx`의 `activeTab` 상태와 `tabs` 배열로 화면을 전환한다. 확인 결과 나눔 탭은 `ShareScreen` 컴포넌트가 정의되어 있었지만 실제 렌더 분기와 하단 탭 배열에서 빠져 있어 진입 시 화면이 비거나 진입 자체가 불가능한 상태였다. 또한 dev 서버에서는 `HospitalReviewItem` 함수 본문이 깨져 OXC 파서 오류가 발생했고, 별도 `src/components/my-pet/PetsScreen.tsx` 파일도 깨진 JSX 때문에 전체 빌드가 막히고 있었다.

## 수정 파일

- `src/App.tsx`: 나눔 탭 연결, 나눔 작성 플로우 연결, 프로필/임시저장에서 나눔 글 열기·수정 연결, 깨진 병원 리뷰 카드 함수 복구, 마이펫 컴포넌트 import 복구
- `src/components/my-pet/PetsScreen.tsx`: 깨진 JSX 파일을 정상 마이펫 목록 컴포넌트로 복구

## 핵심 변경 내용

- 모바일 하단 내비바 5개 기준에 맞춰 `share` 탭을 다시 추가했다.
- `activeTab === 'share'`일 때 `ShareScreen`이 렌더되도록 연결했다.
- 나눔 탭의 `+` 버튼이 QNA 작성이 아니라 나눔 작성 플로우를 열도록 수정했다.
- 프로필의 나눔 글/임시저장 항목이 준비 중 메시지로 막히지 않고 나눔 화면 또는 나눔 작성 화면으로 이어지도록 수정했다.
- `HospitalReviewItem`의 깨진 JSX를 병원 리뷰 카드 UI로 복구해 dev 서버 흰 화면 원인을 제거했다.
- 빌드 검사를 막던 별도 마이펫 컴포넌트 파일을 정상 JSX로 복구했다.
