# maps 작업 기록

## 역할

지도, 병원 검색, 위치 기반 지도, 병원 정보, 리뷰 작성/조회, 병원 좋아요, 리뷰 좋아요 관련 작업을 기록한다.

## 기준

- 지도 화면은 전체 화면 지도를 기본으로 한다.
- 검색창과 분류 태그는 지도 위 오버레이로 둔다.
- 병원 리뷰는 외부 사이트로 이동하지 않고 앱 내부에서 열려야 한다.
- 리뷰 작성은 마이펫, 캘린더 기록, 병원 데이터와 최종적으로 연결되어야 한다.

- 지도/QNA 공용 병원 검색 결과에서 일반 상점, 카페, 펫샵, 용품점, 호텔 등이 섞이지 않도록 프론트 필터를 추가했다.

- ASCII 문맥 기준 패치와 Node.js 안전 치환으로 기본 검색어를 복구했다.
