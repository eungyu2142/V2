# UI 작업 기록

## 역할

색상, 레이아웃, 컴포넌트 모양, 아이콘, 버튼, 카드, 지도 오버레이, 하단 내비게이션 등 시각 UI 변경 사항을 기록한다.

## 기준

- 앱 UI는 민트/아이보리 기반의 깔끔한 스타일을 유지한다.
- 네이버 지도 UX는 참고하되, 앱 자체 UI 정체성은 특수동물 케어 PWA에 맞춘다.
- 화면 요소가 웹 브라우저와 모바일 화면에서 깨지지 않아야 한다.

## 2026-07-03 기록

### 문제 인식

사용자는 UI 관련 작업 결과를 코드와 별도로 계속 추적할 수 있는 기록 파일을 요청했다.

### 해결 과정

`docs/project-records/UI.md`를 만들고, 이후 UI 수정 프롬프트를 수행할 때마다 문제 인식, 해결 과정, 완료 기록을 이 파일에 누적하도록 기준을 세웠다.

### 완료 기록

UI 작업 기록 파일을 생성했다.
