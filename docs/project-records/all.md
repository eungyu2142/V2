# all 전체 화면 작업 기록

## 역할

전체 화면, PWA 공통 동작, 웹 브라우저 화면, 모바일 화면, 반응형 레이아웃, 공통 컴포넌트 수정 사항을 기록한다.

## 기준

- PWA 기반이므로 컴포넌트를 수정할 때 웹 브라우저 화면과 모바일 화면을 함께 고려한다.
- 화면이 깨지지 않도록 전체 화면 기준과 모바일 기준을 같이 확인한다.
- 여러 기능에 영향을 주는 변경은 이 파일에 함께 기록한다.

## 2026-07-03 기록

### 문제 인식

사용자는 `all`을 전체 화면 기록 파일로 두고, PWA 기반에서 웹 브라우저 화면과 모바일 화면을 같이 수정하고 화면이 깨지지 않게 관리하길 강조했다.

### 해결 과정

`docs/project-records/all.md`를 만들고, 기능별 파일에 들어가기 어려운 공통 반응형/전체 화면/공통 컴포넌트 작업을 이 파일에 누적하도록 했다.

### 완료 기록

전체 화면 및 PWA 공통 작업 기록 파일을 생성했다.
