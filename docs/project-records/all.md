## 2026-07-23 작성 화면 모바일 뒤로가기 및 선택 UI 개선

### 요청
모바일 기기 자체 뒤로가기에서 작성 이전 단계로 이동하고, 작성 버튼을 `이전/다음`으로 정리하며 임시저장을 작은 버튼으로 이동. 긴 종 선택 목록을 축소하고 기타 선택 시 직접 입력을 제공.

### 수정 내용
- `StepShell`과 다이어리 작성 화면에 브라우저 `popstate` 처리를 추가해 단계 이전 이동을 지원.
- 작성 단계 하단을 이전/다음 또는 이전/등록 구조로 통일하고 임시저장은 우측 상단 작은 버튼으로 분리.
- 선택 버튼의 모바일 높이와 글자 크기를 축소하고 2열 배치를 적용.
- 파충류의 `기타 도마뱀`, `기타 뱀`을 제거.
- 모든 동물 분류에 `직접 입력(기타)`를 추가하고 선택 시 종 입력칸을 표시.

### 수정 파일
- `src/App.tsx`
- `src/App.css`
- `src/features/diary/DiaryPage.tsx`
- `src/features/diary/DiaryPage.css`
