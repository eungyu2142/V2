# Q&A 카드 첨부 미리보기

## 요청 요약

- Q&A 목록 카드에 사진이나 기록 시각화가 있으면 오른쪽에 미리 보여준다.

## 분석 및 판단

- 기존 카드에는 첨부 여부 텍스트만 있었으므로, 기존 데이터 구조를 유지한 채 `post.image`, `attachedDiarySnapshot`, `attachedRecordSnapshot`을 카드 미디어 영역에 연결했다.
- 실제 사진은 원본 썸네일을 사용하고, 기록은 첨부된 기록 유형별 개수를 기반으로 작은 막대형 미리보기를 표시했다.

## 수정 파일

- `src/App.css`
- `docs/project-records/2026-07-23-qna-card-attachment-preview.md`

## 핵심 변경 내용

- 질문 카드 본문을 텍스트 영역과 오른쪽 미디어 영역으로 분리했다.
- 사진 첨부 시 92px 썸네일을 표시한다.
- 다이어리 기록 첨부 시 기록 유형별 개수를 작은 시각화 막대로 표시한다.
- 단일 기록 첨부 시 기록 유형을 시각화 미리보기의 제목으로 표시한다.
- 360px 이하에서는 미디어 영역을 76px로 줄여 질문 가독성을 유지한다.
- 첨부가 없는 카드에는 미디어 영역을 표시하지 않는다.
