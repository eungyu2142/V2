# share 작업 기록

## 역할

나눔 목록, 나눔 글 작성, 카테고리, 사진/동영상 첨부, 좋아요, 임시저장 관련 작업을 기록한다.

## 기준

- 나눔 글 목록은 최신순이다.
- 목록에서는 머리글과 첨부 사진이 보여야 한다.
- 좋아요와 임시저장은 프로필의 나의 활동과 연결되어야 한다.
