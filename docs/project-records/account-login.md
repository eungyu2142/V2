# account/login 작업 기록

## 역할

로그인, 회원가입, 로그아웃, 계정 생성, 계정 삭제 등 계정 관련 작업을 기록한다.

## 기준

- 계정 기능은 프로필의 계정 관리와 연결되어야 한다.
- 로그인 상태에 따라 좋아요, 임시저장, 내가 쓴 글/리뷰, 반려동물 기록이 자연스럽게 연결되어야 한다.

## 2026-07-03 기록

### 문제 인식

사용자는 account/login 범주를 별도 기록 파일로 분리하길 요청했다.

### 해결 과정

### 완료 기록
